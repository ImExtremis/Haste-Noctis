# Noctis Linux metadata
These files are for Linux desktop integration and package managers/app stores.
## Files
- `app.noctis.Noctis.desktop` - XDG desktop entry for app launchers
- `app.noctis.Noctis.metainfo.xml` - AppStream metadata for package manager and app store listings
- `app.noctis.Noctis.svg` - Application icon

|-------|------|-------------|

| endpoints | [WellKnownNoctisResponseEndpoints](#wellknownnoctisresponseendpoints) | Endpoint URLs for various services |

| oauth2? | [WellKnownNoctisResponseOauth2](#wellknownnoctisresponseoauth2) | OAuth2 endpoints for federation |
| public_key? | [WellKnownNoctisResponsePublicKey](#wellknownnoctisresponsepublickey) | Public key for E2E encryption |
| push | [WellKnownNoctisResponsePush](#wellknownnoctisresponsepush) | Push notification configuration |

<a id="wellknownnoctisresponsefeatures"></a>

## WellKnownNoctisResponseFeatures

Feature flags for this instance

| Field | Type | Description |
|-------|------|-------------|
| manual_review_enabled | boolean | Whether manual review mode is enabled for registrations |
| self_hosted | boolean | Whether this is a self-hosted instance |
| sms_mfa_enabled | boolean | Whether SMS-based MFA is available |
| stripe_enabled | boolean | Whether Stripe payments are enabled |
| voice_enabled | boolean | Whether voice/video calling is enabled |

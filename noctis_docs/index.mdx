Noctis is a free and open source instant messaging and VoIP platform built for friends, groups, and communities.

With Noctis's HTTP, WebSocket Gateway, and Media Proxy APIs, you can build integrations on the Noctis platform, whether you're creating a bot, a webhook, or an OAuth2 integration.

Noctis's APIs are consistent across Noctis instances, as long as they're running the latest Noctis Server version. This applies whether you use the official noctis.app deployment operated by Haste Industries or a self-hosted deployment. Just make sure you use the correct base URLs for the instance you're targeting.

Noctis's API surface is intentionally similar to Discord's, but it's not Discord. You should expect some differences. That said, if you've worked with Discord's APIs before, you'll likely feel at home. The part most likely to remain compatible is the core WebSocket Gateway protocol, which means you can often reuse WebSocket connection code from existing Discord libraries.

## Documentation scope

These docs are more comprehensive than you might be used to from Discord since they cover not only bot-facing APIs, but also instance management APIs (since you can self-host Noctis) and client-specific APIs (since Noctis encourages you to build third-party clients). If you're just looking to build a bot, you can safely skip over sections that don't interest you.
